Quando uma feature for priorizada para execucao:

2. Definir sprint no ROADMAP.md
3. Iniciar desenvolvimento com TDD (RED -> GREEN -> REFACTOR)
4. Ao concluir: mover para `../history/YYYY-MM/` e atualizar ROADMAP.md
