## Como priorizar uma ideia

Quando uma ideia for priorizada:

1. Criar arquivo `nome-da-feature.md` em `../features/`
2. Detalhar requisitos, tarefas tecnicas e criterios de aceite
3. Definir sprint no ROADMAP.md
4. Mover para `../current/` ao iniciar execucao
5. Remover da lista acima
